3rd party Perl packages are used in their pure Perl form and stripped
of installation components in order to simplify installation for CASL
applications.

The full original 3rd party packages can be downloaded from:

YAML (version 0.84):
http://search.cpan.org/~mstrout/YAML-0.84/

List::MoreUtils::PP:
http://search.cpan.org/~rehsack/List-MoreUtils-0.413/lib/List/MoreUtils/PP.pm
The package PP.pm has been modified to export needed functions.

Data::Types:
http://search.cpan.org/~dwheeler/Data-Types-0.09/lib/Data/Types.pm

Clone::PP:
http://search.cpan.org/~neilb/Clone-PP-1.06/lib/Clone/PP.pm

List::Maker:
http://search.cpan.org/~dconway/List-Maker-0.005/lib/List/Maker.pm
